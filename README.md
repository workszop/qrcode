# QR Code Generator

A simple web application to generate QR codes for URLs and text.

## Features

- Generate QR codes from any URL or text
- Clean, modern user interface
- Download QR codes as PNG images
- Responsive design that works on all devices
- High error correction level for better scanning

## Usage

1. Open `index.html` in your web browser
2. Enter a URL or text in the input field
3. Click "Generate QR Code" or press Enter
4. Download the QR code using the "Download QR Code" button
5. Click "Clear" to generate a new QR code

## Technologies Used

- HTML5
- CSS3 (with gradient backgrounds and animations)
- JavaScript (ES6+)
- QRCode.js library

## How to Run

Simply open the `index.html` file in any modern web browser. No installation or server setup required!

Alternatively, you can serve it using a local server:

```bash
# Using Python 3
python -m http.server 8000

# Using Node.js http-server
npx http-server
```

Then open `http://localhost:8000` in your browser.
